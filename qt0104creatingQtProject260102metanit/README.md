Конфигурация проекта и файл .pro

.pro — это основной файл конфигурации проекта для системы сборки qmake [1, 2]. В нем перечисляются исходные файлы (SOURCES), заголовочные файлы (HEADERS), используемые модули Qt, библиотеки и другие настройки компиляции [2, 3].
CMakeLists.txt — это аналогичный файл конфигурации, если в качестве системы сборки используется CMake [4, 5].

В современных версиях Qt Creator (начиная с Qt 6) CMake является системой сборки по умолчанию, однако поддержка .pro файлов (qmake) сохраняется для совместимости со старыми проектами [5, 6].
