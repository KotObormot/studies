## Урок №10. Виджеты в Qt5 / Lesson #10: Widgets in Qt5

Виджеты в Qt5 — это «строительные блоки» для создания пользовательского интерфейса. Визуальные объекты на форме, такие как кнопки, метки, поля, меню, раскрывающиеся списки и т.д. — всё это относится к виджетам.

На этом уроке мы рассмотрим следующие виджеты в Qt5:

- QLabel;

- QSlider;

- QComboBox;

- QSpinBox;

- QLineEdit;

- QMainWindow.

Widgets in Qt5 are the building blocks for creating a user interface. Visual objects on a form, such as buttons, labels, fields, menus, drop-down lists, and so on, are all widgets.
