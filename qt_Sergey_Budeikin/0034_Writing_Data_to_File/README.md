In this video tutorial, we'll use a practical example to demonstrate how to write data to a file from our program.

Для записи данных в файл, мы сначала открываем файл в режиме записи, затем создаем поток вывода, направленный в файл, и используем оператор << для записи данных в этот поток. В примере, приведенном ниже, имена пяти дистрибутивов Linux записываются в файл C:\distros.txt

To write data to a file, we first open the file in write mode, then create an output stream directed to the file, and use the << operator to write data to that stream. In the example below, the names of five Linux distributions are written to the file C:\distros.txt
