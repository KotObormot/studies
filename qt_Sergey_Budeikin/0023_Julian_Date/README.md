In this video tutorial, we'll calculate several historical dates using the Julian date in days, or more precisely, how many Julian days have passed from a given event to the current day.

Юлианский период (не путать с юлианским календарем) начался 1 января 4713 года до нашей эры.
Суть: Это хронологический цикл, разработанный Иосифом Скалигером в XVI веке, охватывающий 7980 лет.
Цель: Создание непрерывного счета дней для астрономии и хронологии, чтобы избежать путаницы с эрами «до нашей эры» и «нашей эры».
Название: Период назван «юлианским» либо в честь Юлия Цезаря, либо (что более вероятно) в честь отца Скалигера, Юлия Цезаря Скалигера.
Завершение: Этот 7980-летний цикл закончится 31 декабря 3267 года.

Важные замечания
Только дата: Метод toJulianDay() работает только с QDate, поэтому при использовании QDateTime необходимо сначала вызвать .date().
Точность: Юлианский день — это целое число (количество дней), оно не включает информацию о времени (часы, минуты, секунды).
Тип данных: Используйте qint64 для хранения юлианских дней, так как числа могут быть большими.
Modified Julian Day (MJD): Если вам нужен Модифицированный юлианский день (начинается с 17.11.1858), вычтите из обычного JD число 2400000.5

Для расчета интервала (количества дней) между двумя датами в Qt (C++) используется метод daysTo() класса QDate.

Альтернатива: 
QDateTime (для разницы в секундах/минутах)
Если вам нужно посчитать интервал не только в днях, но и в часах/минутах/секундах, 
используйте класс QDateTime и его метод secsTo():

QDateTime dateTime1 = QDateTime::currentDateTime();
QDateTime dateTime2 = dateTime1.addDays(1).addHours(2);

qint64 seconds = dateTime1.secsTo(dateTime2);
qint64 minutes = seconds / 60;
qint64 hours = minutes / 60;

qDebug() << "Секунд:" << seconds;
